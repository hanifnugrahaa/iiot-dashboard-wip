# Dashboard Hiliriset IIOT 2026

Welcome to the Industrial IoT (IIOT) Dashboard! This is a Next.js web application built to display real-time weather and environment data across multiple cities.

🌍 **Live Demo:** [https://iiot-dashboard-2026.vercel.app](https://iiot-dashboard-2026.vercel.app)

## Tech Stack
- **Framework**: Next.js 16 (App Router)
- **UI Library**: React 19
- **Styling**: Tailwind CSS v4 (Glassmorphism Design)
- **API**: Open-Meteo API (Server-side fetched)

## Getting Started

First, install the dependencies (if you haven't):
```bash
npm install
```

Then, run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result. You can start editing the page by modifying `app/page.tsx`.
